## 3. Paleta de Cores Definida

### Decisao Final
Baseado em pesquisa de psicologia de cores para construcao e tendencias 2026:

| Uso | Cor | Hex |
|---|---|---|
| Background principal | Charcoal / Dark Gray | `#1A1A1A` |
| Background secundario | Gray escuro | `#2D2D2D` |
| Texto principal | Branco | `#FFFFFF` |
| Texto secundario | Gray claro | `#B0B0B0` |
| CTA / Accent | Amber forte | `#F59E0B` |
| CTA hover | Laranja | `#F97316` |

### Justificativa
- **Amarelo/Amber:** remete a equipamento pesado, cautela, construcao
- **Fundo escuro:** cria contraste maximo com CTA = mais conversoes
- **Hover laranja:** feedback visual forte no botao
- **Charcoal (nao preto puro):** mais sofisticado, menos agressivo

## 4. Melhores Praticas — Landing Pages para Google Ads

### Estrutura de Alta Conversao
1. **Headline = keyword do anuncio** (match exato com o que o usuario buscou)
2. **Formulario acima do fold** (visivel sem scroll)
3. **Sem menu de navegacao** nas landing pages (so CTA ou sair)
4. **Telefone clicavel** visivel no topo
5. **Uma pagina = um servico = um CTA**

### Performance
- Pagina deve carregar em **menos de 3 segundos** no mobile
- Landing page dedicada converte **3-5x mais** que homepage generica
- Taxa de conversao esperada para servicos locais: **5-15%** com pagina otimizada

### Elementos de Confianca (Trust Signals)
- Veteran-owned (diferencial do cliente)
- Licensed, bonded, insured
- Anos de experiencia
- Fotos reais de projetos
- Reviews/testimonials

## 6. Estrutura do Site Planejada

```
/ (Main Page) — visao geral, links para servicos
/excavation — landing page dedicada
/grading — landing page dedicada  
/retaining-walls — landing page dedicada
```

Cada landing page segue a estrutura:
1. Hero (headline + CTA + formulario)
2. Trust signals
3. Servico detalhado
4. Problemas que resolve
5. Processo (4 passos)
6. Galeria
7. CTA final
8. FAQ
